## üîß Customization

### Adding New AI Sites
1. Open `index.html`
2. Find the `aiSites` array in the JavaScript section
3. Add your new site object:

```javascript
{
    name: "Your AI Site",
    description: "Description of the AI site",
    url: "https://example.com",
    category: "conversational", // or other category
    pricing: "free", // free, premium, or enterprise
    features: ["Feature 1", "Feature 2"],
    logo: "ü§ñ" // emoji or icon class
}
```

### Changing Colors
Modify the CSS custom properties in the `:root` selector:

```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
    --accent-color: #your-color;
}
```
